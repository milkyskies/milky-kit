# Capacitor setup

This app is configured to wrap with Capacitor for iOS + Android. The native projects are NOT generated by the scaffold — you initialize them once after install.

## First-time setup

```bash
cd apps/{{app_name}}
pnpm install

# Build the web app (Capacitor needs `dist/` to exist before adding platforms)
pnpm build

# Add native platforms
pnpm cap add ios
pnpm cap add android

# Open in Xcode / Android Studio
pnpm ios:open
pnpm android:open
```

## Day-to-day

| Task | Command |
|---|---|
| Web dev (browser) | `pnpm dev` |
| Sync web build into native projects | `pnpm cap:sync` (runs automatically on `pnpm build`) |
| Run on iOS simulator | `pnpm ios:run` |
| Run on Android emulator | `pnpm android:run` |

## Live updates (OTA)

Capacitor's killer feature is shipping JS updates without app store review. Look at:
- [Capgo](https://capgo.app) — open-source self-hostable OTA updater
- [Ionic Appflow](https://ionic.io/appflow) — managed OTA

Add the chosen plugin and configure per its docs — not part of the scaffold by default.
