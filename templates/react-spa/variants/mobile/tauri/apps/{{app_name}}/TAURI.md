# Tauri setup

This app is configured to wrap with Tauri 2 for desktop + mobile. The `src-tauri/` Rust crate is NOT generated by the scaffold — initialize it once after install.

## First-time setup

```bash
cd apps/{{app_name}}
pnpm install

# Initialize the Tauri Rust crate. Defaults are fine; just confirm prompts.
pnpm tauri init

# Run the desktop app
pnpm tauri:dev

# Run on iOS / Android (requires Xcode / Android SDK)
pnpm tauri:ios:dev
pnpm tauri:android:dev
```

## Day-to-day

| Task | Command |
|---|---|
| Web dev (browser only, no native shell) | `pnpm dev` |
| Desktop dev | `pnpm tauri:dev` |
| Build desktop | `pnpm tauri:build` |
| Build mobile | `pnpm tauri:ios:dev` / `tauri:android:dev` |

After `pnpm tauri init` you'll have `src-tauri/` checked into the repo. Edit `src-tauri/tauri.conf.json` to set the app identifier, window title, etc.
